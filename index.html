Correcting Rural Building Annotations in OpenStreetMap using Convolutional Neural Networks

Proposed a methodology to correct building footprint annotations in OpenStreetMap using Markov Random Fields, Convolutional Neural Networks and shape priors. Our proposed methodology improved in more than 50% the accuracy of the original OSM building annotations.

Fine-grained landuse characterization using ground-based pictures: a deep learning solution based on globally available data

Proposed a method to predict the landuse of urban footprints in OpenStreetMap using ground-based pictures from multiple views of the same urban object. The method uses a Siamese Convolutional Neural Network to perform feature extraction from the multi-view images and it outperformed by 12% of accuracy state-of-the-art baseline methods.

[View More]

Behavior Knowledge Space-Based Fusion for Copy-Move Forgery Detection

Proposed a technique to detect copy-move image tampering. Copy-move forgery is one of the main methods of image forgery that consists in selecting, copying and pasting some regions from and to an image duplicating or hiding objects of regions of interest. Our proposed method is 5% more accurate than other state-of-the-art methods including fusion-based techniques.

An Iterative Spanning Forest Framework for Superpixel Segmentation

A superpixel is a region of similar and connected pixels, that usually belong to the same object. Superpixel segmentation has several applications like in multi-class object segmentation, object detection, depth estimation, and target tracking. We propose a framework for graph-based superpixel segmentation in 2D and 3D images that attain state-of-the-art performance in several image datasets.
